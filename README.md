# Magento 2 Honeypot 

Module for Magento 2 that prevents form spam through a honeypot field.  

## Requirements

* Magento 2 Open Source 2.4.x 
* Adobe Commerce (Magento 2 Enterprise Edition) 2.4.x

## Configuration

Module configuration can be found in the backend: 'Stores' > 'Configuration' > 'Advanced' > 'System' > 'Honeypot'
